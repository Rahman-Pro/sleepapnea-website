# SleepApnea Center Website

A complete, dynamic, professional, and responsive **Sleep Apnea** informational and medical website built with pure HTML5, CSS3, and Vanilla JavaScript.

## 🏥 Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero section, statistics, testimonials, and CTAs |
| About Us | `about.html` | Mission, team profiles, and core values |
| Services | `services.html` | 8 medical services with descriptions |
| Articles | `articles.html` | Blog articles with search and filter |
| News | `news.html` | Dynamic news feed with categories |
| FAQ | `faq.html` | Accordion FAQs with search functionality |
| Reports | `reports.html` | Research reports and statistics with charts |
| Contact | `contact.html` | Validated contact form and location info |

## 🛠 Tech Stack

- **HTML5** — Semantic markup, accessibility attributes
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — Dynamic rendering, form validation, interactivity
- **Google Fonts** — Inter and Poppins typefaces

## 🚀 How to Run

Simply open `index.html` in any modern web browser — no build tools or server required!

```bash
# Clone the repository
git clone https://github.com/Rahman-Pro/sleepapnea-website.git

# Open in browser
open sleepapnea-website/index.html
```

## ✨ Features

- 🎨 **Modern design** with calming medical color palette (blues, teals, whites)
- 📱 **Fully responsive** — mobile-first with hamburger navigation
- ⚡ **Dynamic content** — articles, news, FAQs rendered from JavaScript data
- 🔍 **Search & filter** — on Articles and FAQ pages
- 🎯 **Form validation** — real-time validation on Contact page
- 🔝 **Back-to-top** button with smooth scroll
- ♿ **Accessible** — semantic HTML, ARIA labels, keyboard navigation
- 🌐 **Zero dependencies** — no frameworks, no build tools
